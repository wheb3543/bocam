## نظرة عامة على النظام

منصة إدارة حملات التسويق للمستشفى السعودي الألماني هي نظام متكامل تم تطويره باستخدام أحدث التقنيات الويب. تجمع المنصة بين واجهة عامة احترافية للعملاء ولوحة تحكم متقدمة للإدارة والفرق.

### الأهداف الرئيسية

يهدف النظام إلى تحقيق الأهداف التالية:

- **تحسين تجربة المريض:** توفير طريقة سهلة وسريعة للحجز والتسجيل
- **تحسين الكفاءة الإدارية:** أتمتة العمليات الإدارية وتقليل الأعمال اليدوية
- **تحسين التواصل:** توفير قنوات متعددة للتواصل مع العملاء
- **تحليل البيانات:** جمع وتحليل البيانات لتحسين الخدمات
- **إدارة الفرق:** تنظيم وإدارة فرق التسويق المختلفة

## المكونات الرئيسية

### 1. الواجهة العامة (Frontend)

الواجهة العامة مبنية باستخدام React 19 وTailwind CSS 4، وتتضمن:

| المكون | الوصف |
|---|---|
| **صفحات عامة** | الصفحة الرئيسية، الأطباء، العروض، المخيمات |
| **نماذج الحجز** | نماذج حجز المواعيد والعروض والمخيمات |
| **لوحة التحكم** | واجهة إدارية متقدمة للموظفين |
| **الاستجابة** | تصميم responsive يعمل على جميع الأجهزة |
| **PWA** | تطبيق ويب تقدمي قابل للتثبيت |

### 2. الخادم (Backend)

الخادم مبني باستخدام Express 4 وNode.js، ويوفر:

| الخدمة | الوصف |
|---|---|
| **tRPC API** | واجهة برمجية آمنة وموثوقة |
| **المصادقة** | نظام مصادقة آمن باستخدام JWT |
| **قاعدة البيانات** | MySQL/TiDB لتخزين البيانات |
| **الإشعارات** | نظام إشعارات فوري |
| **التكاملات** | تكاملات مع خدمات خارجية (Meta, Telegram) |

### 3. قاعدة البيانات

قاعدة البيانات تتضمن الجداول التالية:

| الجدول | الوصف | عدد الحقول |
|---|---|---|
| **users** | بيانات المستخدمين | 7 |
| **doctors** | بيانات الأطباء | 12 |
| **appointments** | مواعيد الأطباء | 10 |
| **offers** | العروض المتاحة | 8 |
| **offerLeads** | حجوزات العروض | 6 |
| **camps** | المخيمات | 8 |
| **campRegistrations** | تسجيلات المخيمات | 6 |
| **campaigns** | الحملات التسويقية | 8 |
| **tasks** | مهام الفرق | 10 |

## الصفحات والميزات

### الصفحات العامة (للعملاء)

#### 1. الصفحة الرئيسية
- **المسار:** `/`
- **الميزات:** عرض خدمات المستشفى، روابط سريعة، معلومات الاتصال
- **الجمهور:** جميع الزوار

#### 2. صفحات الأطباء
- **المسار:** `/doctors`, `/doctors/:slug`, `/visiting-doctors`
- **الميزات:** عرض قائمة الأطباء، البحث والفلترة، حجز المواعيد
- **البيانات:** أكثر من 50 طبيب

#### 3. صفحات العروض
- **المسار:** `/offers`, `/offers/:slug`
- **الميزات:** عرض العروض المتاحة، التفاصيل الكاملة، التسجيل
- **البيانات:** عروض متعددة

#### 4. صفحات المخيمات
- **المسار:** `/camps`, `/camps/:slug`
- **الميزات:** عرض المخيمات، التفاصيل، التسجيل
- **البيانات:** مخيمات متعددة

#### 5. نموذج حجز المواعيد
- **المسار:** `/book-appointment`
- **الميزات:** حجز موعد مع الطبيب، اختيار الإجراء، إدخال البيانات
- **التحقق:** التحقق من صحة البيانات

### لوحة التحكم الإدارية

#### 1. الصفحة الرئيسية
- **المسار:** `/dashboard`
- **الميزات:** لمحة عامة، إحصائيات، تبويبات متعددة
- **الصلاحيات:** مسؤول فقط

#### 2. إدارة الأطباء
- **الميزات:** إضافة، تعديل، حذف، تفعيل/تعطيل
- **البيانات:** معلومات شاملة عن الأطباء
- **الإحصائيات:** عدد الأطباء، الأطباء الزائرين، الحالة

#### 3. إدارة المواعيد
- **الميزات:** عرض المواعيد، تعديل الحالة، إرسال تنبيهات
- **الفلترة:** حسب الطبيب، التاريخ، الحالة
- **الإجراءات:** اتصال، WhatsApp، تصدير

#### 4. إدارة العروض والمخيمات
- **الميزات:** إضافة، تعديل، حذف، تفعيل/تعطيل
- **الإحصائيات:** عدد العروض، الحجوزات، الإيرادات
- **الفلترة:** حسب الحالة، النوع

#### 5. إدارة المستخدمين
- **المسار:** `/dashboard/users`
- **الميزات:** إضافة مستخدم، تعديل الصلاحيات، تعطيل الحساب
- **الأدوار:** مسؤول، موظف

#### 6. إدارة المحتوى
- **المسار:** `/dashboard/content`
- **الميزات:** تعديل النصوص، إدارة الصور، إدارة الروابط
- **الأنواع:** نصوص، صور، روابط

#### 7. النشر والمراجعة
- **المسار:** `/dashboard/publishing`
- **الميزات:** مراجعة المحتوى، جدولة النشر، الموافقة
- **سير العمل:** مسودة → مراجعة → نشر

#### 8. التقارير والتحليلات
- **المسار:** `/dashboard/reports`, `/dashboard/analytics`
- **الميزات:** تقارير شاملة، رسوم بيانية، KPIs
- **البيانات:** حجوزات، مواعيد، عروض، مخيمات

#### 9. إدارة الرسائل
- **المسار:** `/dashboard/messages`, `/dashboard/whatsapp`
- **الميزات:** إدارة الرسائل، قوالب الرسائل، السجل
- **القنوات:** البريد الإلكتروني، WhatsApp

### صفحات إدارة الفرق

#### 1. إدارة الحملات والمشاريع
- **المسار:** `/dashboard/projects`
- **الميزات:** عرض الحملات، الإحصائيات، الفلترة

## الأمان والحماية

### نظام المصادقة

- **نوع المصادقة:** OAuth 2.0 مع JWT
- **جلسات آمنة:** استخدام cookies آمنة
- **تشفير:** تشفير كلمات المرور باستخدام bcrypt

### الصلاحيات والأدوار

| الدور | الصلاحيات |
|---|---|
| **زائر** | عرض المعلومات، الحجز |
| **موظف** | إدارة محدودة |
| **مسؤول** | إدارة كاملة |

### حماية البيانات

- **HTTPS:** تشفير جميع الاتصالات
- **CORS:** حماية من الطلبات غير المصرح بها
- **SQL Injection Prevention:** استخدام parameterized queries
- **XSS Prevention:** تنظيف جميع المدخلات

### النسخ الاحتياطية

- **التكرار التلقائي:** نسخ احتياطية يومية
- **الاسترجاع:** إمكانية استرجاع البيانات في حالة الطوارئ

## الخلاصة

منصة إدارة حملات التسويق للمستشفى السعودي الألماني هي نظام متقدم وشامل يجمع بين أفضل الممارسات في تطوير الويب الحديث. توفر المنصة تجربة مستخدم ممتازة للعملاء والموظفين، مع توفير أدوات إدارية قوية للإدارة.

### النقاط الرئيسية

- **تقنيات حديثة:** استخدام أحدث التقنيات والأطر العمل
- **أمان عالي:** نظام أمان متقدم وحماية البيانات
- **قابلية التوسع:** تصميم يسمح بالتوسع المستقبلي
- **تجربة مستخدم ممتازة:** واجهة سهلة الاستخدام وجذابة
- **أداء عالي:** تحسين الأداء والسرعة
