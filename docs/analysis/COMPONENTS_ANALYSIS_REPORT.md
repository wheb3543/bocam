### تحسينات المرحلة 3 على BulkActionsManager ✅:
تم تحسين المكون الجديد BulkActionsManager بإضافة وظائف متقدمة:

**الوظائف الجديدة المضافة:**
1. **أنواع إجراءات جديدة:**
   - `DeleteAction`: حذف جماعي مع حوار تأكيد
   - `ExportAction`: تصدير البيانات مع اختيار الصيغة
   - تحسين `SimpleAction`: إضافة دعم `isLoading`

2. **تحسينات الشريط (Bulk Actions Bar):**
   - `position`: تحديد موقع الشريط (bottom/top/left/right)
   - `size`: تحديد حجم الشريط (compact/normal/large)
   - `barClassName`: تخصيص className للشريط
   - `showCount`: إظهار/إخفاء عدد العناصر
   - `countLabel`: تخصيص نص العنصر المحدد

3. **تحسينات الأزرار:**
   - `className`: تخصيص className لكل زر
   - `disabled`: تعطيل فردي لكل زر
   - `isLoading`: دعم حالة التحميل لجميع أنواع الإجراءات
   - أيقونات تلقائية للحذف (Trash2) والتصدير (Download)

4. **تحسينات الحوار (Dialog):**
   - `dialogTitle`: تخصيص عنوان حوار تحديث الحالة
   - `dialogDescription`: تخصيص وصف حوار تحديث الحالة
   - `confirmTitle`: تخصيص عنوان حوار تأكيد الحذف
   - `confirmDescription`: تخصيص وصف حوار تأكيد الحذف
   - `exportFormats`: خيارات صيغ التصدير

5. **حوارات جديدة:**
   - **Delete Confirmation Dialog**: حوار تأكيد الحذف باستخدام AlertDialog
   - **Export Dialog**: حوار اختيار صيغة التصدير
